# ARC Pay

A decentralized payment application (dApp) for USDC built on the Arc ecosystem.

## Features
- Invoice creation
- QR code payment
- Wallet integration
- Direct deployment on Railway

## Live Demo
https://arc-pay-production.up.railway.app

## Technologies Used
- TypeScript
- JavaScript
- Node.js
- Web3 Wallet Integration
